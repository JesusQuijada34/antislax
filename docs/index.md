## üßê What is AntiSlax?

AntiSlax Pro is born from the need to have a "clean" Windows experience without the overhead of modern telemetry and pre-installed advertising software. It is a lightweight but powerful script that leverages Python and the Windows Registry API to transform your OS.

## üéØ Our Goals

-   **Performance**: Minimize background processes.
-   **Privacy**: Strip away tracking and data collection.
-   **User Control**: Give the power back to the user to choose their installed apps.
-   **Aesthetics**: Provide a high-quality UI that feels native to a premium OS.
