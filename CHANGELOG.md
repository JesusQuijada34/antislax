# Historial de Cambios de Winslax

## [1.1.0] - 2025-10-25
### Añadido
- **Widget de Monitoreo Inteligente:** Se agregó `Winslax_Widget.py`, un script en Python que monitorea el uso de CPU y RAM, identifica procesos de alto consumo y permite terminarlos. También incluye una función de limpieza de archivos temporales.
- **Estructura:** Se renombró la rama principal de `master` a `main` en GitHub.
- **Documentación:** Se creó este archivo `CHANGELOG.md`.

### Cambiado
- **Documentación:** Se actualizó `README.md` para incluir la información sobre el nuevo Widget de Monitoreo.

## [1.0.0] - 2025-10-25
### Añadido
- **Optimización Principal:** Script `Winslax.ps1` para optimización de Windows.
- **Funcionalidad:** Eliminación de bloatware (AppX), desactivación de telemetría, optimización de servicios y ajustes de rendimiento (plan de energía y hibernación).
- **Lanzador:** Archivo `Lanzar_Winslax.bat` para ejecución con permisos de Administrador.
- **Documentación:** Archivo `README.md` inicial.
